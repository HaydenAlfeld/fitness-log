## Project Overview
This repository documents a fitness journey, including strength training routines, weekly workout logs, nutrition planning, and long-term goals.

The purpose of this project is to:
- Track strength progress
- Improve consistency
- Monitor training adjustments
- Reflect on performance and recovery

This repository is organized into the following sections:

- goals/ Long-term fitness goals
- workouts/ Structured training routines
- logs/ Weekly workout summaries
- nutrition/ Meal planning and nutrition notes
- docs/ GitHub Pages site files

This project also demonstrates version control practices using Git and GitHub.
